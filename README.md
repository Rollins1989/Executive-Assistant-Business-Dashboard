# Executive Assistant Business Intelligence Dashboard

## Overview

A multi-page Business Intelligence dashboard developed using Python and Streamlit to automate executive reporting, task management, expense analysis, vendor management, meeting tracking, follow-up monitoring, calendar analytics and MIS reporting.

---

## Features

- Executive Dashboard
- Meeting Analytics
- Task Dashboard
- Follow-up Tracker
- Vendor Management
- Expense Analytics
- Calendar Analytics
- Executive MIS Dashboard
- Interactive Plotly Charts
- KPI Cards
- Google Sheets Integration
- Automated Reports

---

## Technology Stack

- Python
- Streamlit
- Pandas
- Plotly
- NumPy
- OpenPyXL
- Google Sheets API
